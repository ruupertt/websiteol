# Open Legends — RoboCupJunior Vision

A responsive, dependency-free static website for the Open Legends team from Brisbane Boys' College.

## Site sections

- Home
- Team
- Robot and technical documentation
- Achievements
- 2026 international campaign

## Publish with GitHub Pages

1. Create a new GitHub repository, such as `open-legends-2026`.
2. Upload everything in this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and the `/ (root)` folder, then save.
6. GitHub will publish the website at `https://<username>.github.io/<repository-name>/`.

No build step or external packages are required.

## Updating content

- Main text and links: `index.html`
- Colours and layout: `styles.css`
- Navigation, animations and robot system tabs: `script.js`
- Images and team poster: `assets/`

## Before publishing

- Add a campaign or donation URL if one becomes available.
- Confirm all member email addresses should remain public.
- Replace or add high-resolution team and robot photos when available.
- Update achievements after RoboCup 2026.
